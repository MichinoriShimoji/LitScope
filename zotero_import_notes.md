# Zotero import notes for the Irabu dialect bibliography

Files prepared:

- `irabu_dialect_literature.csv`
- `irabu_dialect_literature.bib`
- `pdfs/` contains the open PDFs I could download in this environment.

Important scope note:

- "Core" entries are directly about Irabu Ryukyuan / Irabu Island dialects.
- "Peripheral" entries concern Sarahama/Ikema varieties spoken on Irabu Island.
- The older Motonaga/Nakama/Sawaki/Ryuudaihoogen entries were reconstructed from the reference list in Shimoji (2006), so identifiers such as DOI/ISBN are currently unverified.

Zotero:

- I do not have direct access to the user's Zotero My Library from this Codex environment.
- Import `irabu_dialect_literature.bib` into Zotero via File > Import.
- For PDFs, use Zotero's "Find Available PDFs" after import, or attach the downloaded PDFs from the `pdfs/` directory if present.
